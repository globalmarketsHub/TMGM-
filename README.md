# TMGM Staff Academy

一个用于 TMGM 员工岗前培训的网站原型。

## 演示账号

- 学员：`trainee` / `tmgm2026`
- CRM 管理：`admin` / `tmgm2026`

## 已包含

- 1-8 天培训路线
- 第 1 天产品基础、保证金、合约大小、杠杆、最小最大开仓、隔夜费说明
- 每日例题和小测
- 小测结果同步到本地 CRM 看板
- 在线时间、真实操作时间、活跃度统计
- CRM 数据 JSON 导出
